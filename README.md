*Prompt AI* - AI Based Prompt Optimization 

**Prompt AI** is an intelligent web-based tool designed to enhance and optimize user prompts for AI systems. It helps users generate well-structured, clear, and effective prompts in **Study Mode, Coding Mode, and Career Mode**, improving AI response quality for students, developers, and professionals.

 *Features* 

- **Study Mode:** Generates clear, exam-oriented prompts for learning and revision.  
- **Coding Mode:** Enhances programming-related prompts for debugging, code explanation, and problem-solving.  
- **Career Mode:** Optimizes prompts for resumes, interviews, and career guidance.  
- **Intelligent Prompt Enhancement:** Improves clarity, structure, and intent while preserving original meaning.  
- **User-Friendly Interface:** Simple and intuitive design for beginners and non-technical users.  
- **Instant Output:** Provides enhanced prompts in seconds.  
- **Web-Based Application:** Accessible through any browser without installation.

---

 *Target Users* 

- Students and learners  
- Programming learners and developers  
- Job seekers and professionals  
- Beginners and non-technical AI users

---

 *Known Limitations* 

- No user accounts; session-based only.  
- Limited modes: Study, Coding, Career.  
- No personalization or adaptive learning.  
- Some UI sections (Blog Generator, About Us, Contact) are incomplete.  
- No feedback loop to learn from user input or ratings.  
- No rate limiting; basic API integration.  
- No analytics to track prompt performance.  
- Fully dependent on Gemini AI API; offline mode not supported.

---

 *Technology Stack* 

 *Frontend* : React with TypeScript, Vite, TailwindCSS for styling, Framer Motion for animations, and Shadcn/ui for beautiful components.

 *Backend* : Node.js with Express, PostgreSQL (Neon) for the database, and Drizzle ORM for database queries.

 *Key Libraries* : React Query for data fetching, React Hook Form for forms, and Lucide icons.

 *AI* : Gemini AI integration

**Tools:** Visual Studio Code, Git & GitHub, Browser (Chrome)

---

 *How It Works* 

1. User enters a raw prompt into the application.  
2. Selects the desired mode: Study, Coding, or Career.  
3. Prompt AI analyzes and enhances the prompt for clarity, structure, and intent.  
4. User receives an optimized prompt ready for AI interaction.
